# [Eternal Highlander](https://eternalhighlander.com.br/) 🃏
Eternal Highlander é uma variante estratégica do formato Commander de Magic: The Gathering idealizado por Estevan Carvalho Ribeiro focando em ter um maior equilíbrio dos decks.. Este projeto é a Landing Page oficial do formato, desenvolvida para centralizar as regras, facilitar o acesso à calculadora de decks e servir como ponto de partida para a comunidade.

## 🚀 Funcionalidades Atuais

* **Landing Page Responsiva:** Apresentação clara das regras e filosofia do formato.

* **Integração com Calculadora:** Acesso direto à planilha automatizada para validação e criação de decks.

* **Design First:** O projeto foi inteiramente planejado no Figma antes da implementação, garantindo fidelidade visual e melhor UX.

## 🛠️ Tecnologias Utilizadas
Optei por utilizar Vanilla Stack (tecnologias puras) para demonstrar domínio sobre os fundamentos da web e garantir a melhor performance possível:

* **HTML5** (Semântico)

* **CSS3** (Layouts modernos e responsividade)

* **JavaScript** (Manipulação de DOM e interatividade)

## 📈 Roadmap de Evolução
O projeto foi pensado para ser escalável. As próximas etapas incluem:

* **Blog Integrado:** Para notícias, metas de jogo e análise de decks.

* **Web App Dedicado:** Transição da planilha para uma plataforma própria de cálculo de decks (estilo Moxfield), com banco de dados e autenticação.
